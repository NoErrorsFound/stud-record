Practical 10 — Cloud-Based Student Record Management System (AWS EC2)
Step 1 — Launch EC2 Instance

Go to AWS Console → EC2 → Launch Instance

Choose:

Setting	Value
Name	student-system
AMI	Ubuntu
Instance Type	t2.micro
Key Pair	Proceed without key pair
Allow Traffic	Allow HTTP + SSH

Click:

Launch Instance
Step 2 — Connect to EC2

Click:

Connect → EC2 Instance Connect → Connect
Step 3 — Update Ubuntu

Run:

sudo apt update
Step 4 — Install Node.js

Run:

curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -

Then:

sudo apt install -y nodejs

Check version:

node -v
npm -v
Step 5 — Install MongoDB

Run:

sudo apt-get install gnupg curl
curl -fsSL https://pgp.mongodb.com/server-6.0.asc | sudo gpg -o /usr/share/keyrings/mongodb-server-6.0.gpg --dearmor
echo "deb [ arch=amd64,arm64 signed-by=/usr/share/keyrings/mongodb-server-6.0.gpg ] https://repo.mongodb.org/apt/ubuntu jammy/mongodb-org/6.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-6.0.list
sudo apt update
sudo apt install -y mongodb-org
Step 6 — Start MongoDB

Run:

sudo systemctl start mongod
sudo systemctl enable mongod

Check status:

sudo systemctl status mongod
Step 7 — Clone GitHub Project

Run:

git clone YOUR_GITHUB_URL

Example:

git clone https://github.com/mrunalmohite120725/student-management-system.git
Step 8 — Open Project Folder

Run:

cd student-management-system

Check files:

ls
Step 9 — Install Project Dependencies

Run:

npm install
Step 10 — Open Port 5000 in Security Group

AWS Console → EC2 → Security Groups → Inbound Rules → Edit Inbound Rules

Add:

Type	Port	Source
Custom TCP	5000	Anywhere IPv4

Click:

Save rules
Step 11 — Run Project

Run:

node index.js

You should see:

Server started on port : 5000
Connected to MongoDB Server...

DO NOT PRESS CTRL+C.

Keep terminal running.

Step 12 — Open Website

Browser:

http://YOUR_PUBLIC_IP:5000

Example:

http://34.228.82.242:5000
